## 2. 静默超时（SILENT_TIMEOUT）

（待 Phase D 填充：完整定义、判定逻辑、`last_seen IS NULL` 的特殊处理）

**触发条件**：
- 设备最后一次有数据的时间距当前 > 阈值（默认 600 秒）
- 设备从未上报过数据（`last_seen` 为空）则**不触发** —— 避免新接入设备误报

**调优**：
- 阈值通常设为采集周期的 5-10 倍
- 例：5s 周期 → 60-120s；60s 周期 → 600-1200s
