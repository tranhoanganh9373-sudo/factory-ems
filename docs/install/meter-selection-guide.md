# 电表选型清单（≤50 块场景）

> **适用场景**：单厂、多楼层、Modbus RTU 接入、采样 1 min/次、不要求断网零丢
> **配套**：硬件拓扑见 [field-installation-sop.md](./field-installation-sop.md)；EMS 通道配置见 [channel-config-import.json](./channel-config-import.json)

---

## 1. 选型原则（按优先级）

1. **同一总线选同型号** — 不同厂家电气特性差异大，混挂易冲突
2. **优先国产 + 主流型号** — 售后近、寄存器表好查、一线工程师熟
3. **CT 量程留 1.5 倍余量** — 电流互感器二次额定 5 A，选 1.5×峰值负荷
4. **必须有以下接口/能力**：
   - RS-485 物理接口（不是 RS-232！）
   - Modbus RTU 协议（不是 DL/T 645，DL/T 645 EMS 暂不支持）
   - 可设站号（unitId 1–247）
   - 可设波特率（至少支持 9600）
   - 寄存器表公开可下载

## 3. 互感器（CT）选型

电表本身不直接测大电流，需要配 CT 把一次大电流变成二次 5 A：

| 主开关额定 | CT 变比建议 |
|---|---|
| 100 A | 100/5 |
| 200 A | 200/5 |
| 400 A | 400/5 |
| 630 A | 800/5 |
| 1000 A | 1500/5 |
| 1600 A | 2000/5 |

**精度要求**：和电表同级别（电表 1.0 级 → CT 0.5 级；计费级 0.5S → CT 0.2S）。

**穿芯式 vs 开口式**：
- 新装回路 → **穿芯式**（便宜、精度高）
- 改造回路（不能停电）→ **开口式**（贵 30%，但可现场卡接）

## 4. 串口服务器（每楼层 1 台）

| 厂家 | 型号 | 单价（元） | 备注 |
|---|---|---|---|
| **有人物联** | **USR-N510** | 200–300 | 首选，国产稳定，配置 Web UI 友好 |
| 友善之臂 | DTU-N810 | 150–250 | 更便宜，质量稍弱 |
| Moxa（台资） | NPort 5150 | 1200–1500 | 高端项目 / 合规要求严的选这个 |

配置要点：
- 模式：TCP Server
- 监听端口：502（标准 Modbus TCP 端口）
- 串口参数与电表完全一致（波特率 9600、8N1）
- 包间隔 / 超时保留默认即可（一般 50–200 ms）

## 7. 采购前校验清单

下单前确认：

- [ ] 电表寄存器表 PDF 已拿到（找厂家技术发，不要只看说明书）
- [ ] CT 变比已与现场电气工程师确认（看一次开关额定电流）
- [ ] 站号修改方法已确认（按键 / 拨码 / 软件）
- [ ] 串口服务器 IP 段已与厂网管要好（避免和办公网冲突）
- [ ] EMS 服务器位置 + 取电 + 网络已规划

## 8. 已知不兼容清单

EMS 当前不支持以下场景，下单前规避：

| 协议 / 表型 | 说明 |
|---|---|
| DL/T 645 协议表（多数智能电表外观相似但协议不同） | EMS 仅支持 Modbus RTU/TCP；DL/T 645 需另加协议网关 |
| 红外抄表 / IC 卡表 | 无 RS-485 接口，不在采集范围 |
| 部分 老款施耐德 PowerLogic（PM5xxx 之前） | 寄存器映射非标，需定制驱动 |
