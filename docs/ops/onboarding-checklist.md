| 项 | 最低 | 推荐 | 客户实际 |

| CPU | 4 核 | 8 核 | |

| 网卡 | 1×1G | 2×1G（OT/办公分离） | |
| OS | Ubuntu 22.04 / RHEL 8 / Debian 12 | | |
| Docker / Compose | 24+ / v2 | | □ 已装 / □ 未装 |
| 串口（仅 RTU 场景） | USB-RS485 适配器型号 | | `/dev/ttyUSB0` 等 |

**串口 dialout 组：** 用 RTU 时部署用户要加入 `dialout` 组（`usermod -aG dialout ems`）；
否则容器内读串口会被权限拒绝。docker compose 里加 `--device=/dev/ttyUSB0`。

## 4. 三方系统对接

| 项 | 是否需要 | 备注 |
|---|---|---|
| 客户已有数据库 | □ 是 / □ 否 | 是 → 我们仍部署独立 PG，按需做 ETL 同步 |
