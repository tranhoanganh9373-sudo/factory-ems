## 5. 现场数据异常分析

**Channel 1 (MODBUS_TCP, DISCONNECTED):**
```
错误: "readHolding(unit=1, addr=0x9c41, count=1) failed: 
       Executing transaction ... failed (tried 5 times) 
       Socket timeout reading response - Read timed out"
失败次数: 479 (24h) / 成功: 0
平均延迟: 3ms
```

**诊断**：Modbus TCP 地址 0x9c41 (十进制 40001) 的从站不存在或无响应。  
- 5次重试 (ModbusBackoff: 1s+2s+4s+8s+16s = 31s 总重试时间)  
- 每次 ~3ms → 确认是本地 timeout (非网络延迟)  
- Backoff 级联正常 (可验证)

**建议**：检查 `deploy/collector.yml` 中 Channel 1 的 IP/端口/UnitID 配置。
